# Support

## Usage Questions

Use GitHub issues for questions about:

- choosing the right skill
- improving wording or examples
- missing documentation
- packaging or installation problems with published `.skill` files

## Bugs

Open a bug report with reproduction steps, the affected skill path, and the expected behavior.

## Security Issues

Do not use public issues for security concerns. Follow [SECURITY.md](SECURITY.md).

## Feature Requests

Open a feature request issue and explain the user problem the skill should solve.
